# Repository and Pipeline Inventory Template

Excel template for collecting repository, pipeline, and security integration data from development teams.

## Purpose
This template helps collect comprehensive information about:
- Repositories (GitHub, Azure DevOps, etc.)
- Pipelines (YAML and Classic)
- Security integrations (Snyk, SonarQube, Dependabot)
- Branch usage and management
- Product and project mappings

## Files Included
- `Repository_Pipeline_Inventory.xlsx` - Main data collection template
- `Instructions.md` - Detailed guide for teams
- `Sample_Data.xlsx` - Example filled template

## How to Use
1. Download the Excel template
2. Share with development teams
3. Each team fills one row per repository
4. Collect and consolidate responses
5. Use built-in summary sheets for analysis

## Template Sheets
1. **Instructions** - How to fill out the form
2. **Data Collection** - Main data entry sheet
3. **Reference Data** - Dropdown values for validation
4. **Summary Dashboard** - Auto-calculated statistics
5. **Team Summary** - Per-team aggregation
6. **Product Summary** - Per-product aggregation
7. **Action Items** - Auto-generated tasks
